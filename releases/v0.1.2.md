# beBetter v0.1.2

## 변경 사항

- macOS 데스크톱 앱 내 새 버전 확인, 다운로드 및 설치 지원
- 업데이트 화면 내 현재 버전, 새 버전, 확인 상태 및 다운로드 진행률 표시
- 서명 및 Apple 공증을 거친 공식 GitHub Release 기반 macOS 업데이트 제공

## 수정 사항

- 업데이트 재시도 시 진행 상태 즉시 표시
- 업데이트 상태 변경 시에도 주요 버튼 위치 고정
- 업데이트 화면의 불필요한 사이드바 제어 제거 및 내비게이션 버튼 역할별 재구성

## 설치 안내

- 앱 내 업데이트를 지원하는 첫 버전으로, 기존 v0.1.1 사용자의 DMG 1회 수동 설치 필요
- v0.1.2 설치 이후 앱 업데이트 화면을 통한 후속 버전 확인 및 설치 지원
