# beBetter v0.1.5

## 변경 사항

- 설정에서 하루 마감 버튼의 노출 시작 시간과 종료 시간 지정 지원
- 기본 노출 시간을 오후 8시부터 자정까지로 제공하고 변경한 설정을 자동 저장
- 앱 테마에 맞춘 시간 선택기와 `자정까지 (24:00)` 빠른 선택 옵션 제공

## 수정 사항

- 하루 마감 버튼이 설정 시간과 관계없이 항상 표시되던 문제 수정
- 종료 시간을 조정하는 중간 단계에서 잘못된 경고가 반복되던 문제 수정
- 시작 시간보다 빠른 종료 시간을 적용할 때 안내 토스트가 하나만 표시되도록 개선
- 하단 전역 작업 아이콘 사이의 간격이 일정하지 않던 문제 수정
