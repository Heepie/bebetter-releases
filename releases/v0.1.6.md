# beBetter v0.1.6

## 변경 사항

- Windows 데스크톱 앱에서 macOS와 동일한 업데이트 화면 제공
- Windows에서도 업데이트 확인, 다운로드, 설치 흐름 지원
- macOS와 Windows 데스크톱 artifact를 하나의 release로 함께 배포

## 수정 사항

- 플랫폼별 updater 실행 클래스를 분리해 macOS와 Windows 업데이트 동작을 독립적으로 관리
- 업데이트 상태 흐름과 UI를 공통 구조로 정리
- Linux와 웹 환경에서는 지원하지 않는 updater가 실행되지 않도록 방어
